# 게시판 서비스

가장 기본적이고 보편적인 게시판 기능을 둘러볼 수 있는 서비스

## 개발 환경

* Intellij IDEA Ultimate
* Java 17
* Gradle 7.5
* Spring Boot 2.7.0

## 세부 사항
* Spring Boot Actuator
* Spring web
* Spring Data JPA
* Rest Repositories
* Rest Repositories HAL Explorer
* Thymeleaf
* H2 Database
* MySQL Driver
* Lombok
* Spring Boot Devtools
* Spring Configuration processor

그외

* Querydsl
* Bootstrap
* Heroku
